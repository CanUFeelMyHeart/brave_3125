Чтобы увидеть список всех виток в нашем репозитории, используем:
> git branch

Чтобы создать ветку, используем:
> git branch branch_name:
создается ветка с именем **branch_name**

Чтобы увидеть переход к другой ветке, используем:
> git checkout **branch_name**

Чтобы удалить ветку, используем:
>git branch -d **name_branch**

Для клонирования внешнего репозитория на локальный ПК, используем:
> git clone **url-адрес репозитория**

Для получения изменений и слияния с локальной версией, используем:
> git pull

Для отправки локальной версии репозитория на внешний, используем:
> git push
